Ferramentas de Gestão

ByteBabe oferece integração com várias ferramentas GUI:

  • TablePlus

  • DBeaver

  • MongoDB Compass

  • pgAdmin

  • MySQL Workbench

Instalação

# Instalar banco específico bytebabe db install mysql # Instalar múltiplos bancos bytebabe db setup

Configuração

Cada banco pode ser customizado através do arquivo config.yaml:

mysql: port: 3306 version: "8.0" custom_config: "/path/to/my.cnf" postgres: port: 5432 version: "15" custom_config: "/path/to/postgresql.conf"
