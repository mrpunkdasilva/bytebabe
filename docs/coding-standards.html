Shell Script Guidelines

# ✅ Bom function install_package() { local package_name="$1" if ! command -v "$package_name" &> /dev/null; then echo "Instalando $package_name..." fi } # ❌ Evitar function installPackage { packageName=$1 which $packageName > /dev/null if [ $? -ne 0 ]; then echo "Instalando $packageName..." fi }

Nomenclatura

  • Funções: snake_case

  • Variáveis: snake_case

  • Constantes: SCREAMING_SNAKE_CASE

  • Arquivos: kebab-case.sh
