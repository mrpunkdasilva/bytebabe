Volume Management 💾

Comandos Disponíveis

Estilo Verboso

bytebabe docker volumes [comando] [opções]

Estilo Curto

bytebabe docker v [cmd] [opts]

Tipos de Volume

  • 🔧 Local

  • 🌐 NFS

  • ☁️ Cloud Storage

  • 🔒 Encrypted

  • 📦 Plugin Volumes

Exemplos

Estilo Verboso

# Criar volume bytebabe docker volumes create --name dbdata # Listar volumes bytebabe docker volumes list # Remover volume bytebabe docker volumes remove dbdata

Estilo Curto

# Criar volume bytebabe docker v new -n dbdata # Listar volumes bytebabe docker v ls # Remover volume bytebabe docker v rm dbdata
