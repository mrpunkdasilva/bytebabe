Criar Branch

# Interativo bytebabe git branch new # Direto bytebabe git branch new feature/nome

Merge de Branches

# Merge interativo bytebabe git branch merge # Com confirmação visual bytebabe git branch merge develop --visual

# Com validação bytebabe git branch delete feature/old # Força deleção bytebabe git branch delete feature/old --force
