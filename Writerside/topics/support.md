## FAQ

### Instalação
Q: Como atualizar o ByteBabe?
```bash
bytebabe self-update
```

### Configuração
Q: Onde ficam os arquivos de config?
```bash
~/.bytebabe/config/
```

### Troubleshooting
Q: Como gerar relatório de diagnóstico?
```bash
bytebabe doctor --report
```
